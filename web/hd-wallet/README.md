# hd-wallet

   本项目实现一个基于`vue`+`typescript`+`ether.js`简单实现的钱包功能


## 二、功能描述

   * 新建账号
    ...


## 三、使用说明

### 依赖安装
 * 安装`vue3`:`npm install -g @vue/cli`
 * 在项目根目录执行安装本项目依赖的命令：`npm install`

### 启动和访问
 * 启动：`cd hd-wallet && npm run serve`
 * 访问：`http://localhost:8080/`
